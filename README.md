# Password Manager

Password Manager is a full-stack web application that allows users to securely save, manage, and retrieve their passwords. 
Built using the MERN stack (MongoDB, Express.js, React, Node.js), it features a user-friendly interface, secure authentication, 
and CRUD functionality for managing passwords.

## Demo  
[Watch the Demo](https://drive.google.com/file/d/1C97LVa1XJ_vhPaFhf7jN4nkG3thKU4b1/view?usp=sharing)

---

## Features  
- **User Authentication**: Register and log in securely using JWT-based authentication.
- **Password Management**:  
  - Add new passwords.
  - Edit existing passwords.
  - View details of saved passwords.
  - Delete passwords.
- **Responsive Design**: Optimized for all devices using Tailwind CSS.  
- **Secure Storage**: Sensitive information is stored in a secured database.

---

## Technologies Used  
### Frontend  
- **React**: For building the user interface.  
- **Tailwind CSS**: For styling.  
- **Vite**: For fast development and build setup.

### Backend  
- **Node.js**: Backend runtime.  
- **Express.js**: Web framework.  
- **MongoDB**: Database to store user and password data.  
- **JWT**: For secure authentication.  

---

### Prerequisites  
- **Node.js**: Install from [Node.js Official Website](https://nodejs.org).  
- **MongoDB**: Set up a MongoDB instance (local or cloud).
